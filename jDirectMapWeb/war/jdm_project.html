1 - Description:

 

The jQuery Direct Map or jDirectMap is a project to enable users to relate two different structural data sets. To support clear design and optimize performance for hierarchival strcuture, combination of spindown with tree control is used to selectively hide and show information or nodes. The simplest form is a spindown or disclosure triangle which is used to hide or show a single block of information. A set of these stacked together is referred to as a tree control. More information on UI guidelines:

Apple UI guidelines

 

 

Currently the jQuery UI Tree plugin is in development, but not yet documented (as it's not part of a jQuery UI release). is a tree page on the UI documentation wiki page with more details about the project:

jQuery Tree project

 

As part of this project a wide research on currently avaiable jQuery tree plug-ins was conducted. Results (full report to be published shortly) shown that there are fairly robust plugins with lots of features to consider:

DynaTree

jsTree

zTree

 

As part of the project all of these major plugins will be considered and implemented in demo. The plan is to implement basic versions extending each of these tree plugins and leave the selection process to the user. Certain minimal requirements and fuctional specification with precondition need to be defined and followed in each version.
